Galaxy S8 Clock App Alarm Feature

This is probably my most used app all the time. Whether I'm setting an alarm while cooking, or setting an alarm to wake up the next morning, I am always using this app.

My biggest problem that I run into with this app is the scrolling mechanic. I'm always setting up new alarms for random times so I would rather type in a time instead of scrolling to find the correct time.

For my solution, I want to:

  • Allow the user to simply type in a time for the alarm to go off
  • Also allow user to scroll for desired time
  • Insert a "Time's Up!" notification
  • Utilize validation checks
